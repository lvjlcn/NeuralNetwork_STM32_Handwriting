# NeuralNetwork_STM32_Handwriting
A simple neural network run on stm32 F407 using the DSP libaray.

##Hardware:
 CPU: STM32F407VEtx  Flash 512k, RAM 192K,  Freq 168M
 LCD controlor: ili9341 
 Touch: XPT2046

 input Layer  28*28 pixel 784 nodes
 lay2   30 nodes
 lay3    10 nodes
